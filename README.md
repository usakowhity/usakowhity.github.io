\### \*\*PetApp3 Portable\*\*

ユーザー自身の犬・猫・うさぎを登録し、

https://github.com/usakowhity/PetApp3-portable



\### \*\*PetApp4-Web (Myu Edition)\*\*

世界初の「猫語検定」アプリ。  

Myu（Domestic Shorthair）が『猫語検定協会』会長として登場し、  

あなたの声を猫語レベルで判定します。



猫語に近いと好まれ、harsh（大きな音）や乱暴な操作は嫌われ、  

ねこじゃらしにも反応します。  

無操作では「眠る → 背伸び → 平静」に戻る猫らしい挙動をします。



公開ページ：  

https://petapp4-web-myu.netlify.app
